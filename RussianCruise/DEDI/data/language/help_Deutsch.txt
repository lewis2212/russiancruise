\con_wind
^7Schaltet während der Verbindung in den Fenstermodus um

Falls der Computer nicht bereits mit dem Internet verbunden ist, wird oftmals ein Verbindungsfenster angezeigt. Die Umschaltung in den Fenstermodus ermöglicht es die Verbindung herzustellen, ohne LFS beenden zu müssen.

Oftmals wird auch durch ein anderes Programm z.B. eine Firewall eine Meldung angezeigt, was unter bestimmten Umständen sogar zum Absturz führen kann. Dieses Problem kann durch die Aktivierung des Fenstermodus nicht auftreten.

Falls bei der Verbindungsherstellung keine Hinweisfenster erscheinen, kann die Option einfach ausgeschaltet werden.

Weitere Informationen gibt es im Online Handbuch: ^7www.lfsmanual.net^8

\host_ip
^7Gibt die IP Adresse des Servers an

Um sich mit einem bestimmten Computer im LAN oder Internet zu verbinden, ist die Eingabe der IP Adresse des anderen Computers erforderlich. Die IP Adresse ist eine Nummer die aus 4, jeweils durch einen Punkt getrennten, 1 bis 3-stelligen Zahlen besteht. In Heimnetzwerken (LAN) werden meistens IP Adressen in der Form 192.168.0.x verwendet, wobei x für eine beliebige 3-stellige Zahl steht.

Wenn du über das Internet spielst, lässt sich die IP Adresse Deines Computers am einfachsten über die Seite ^3www.whatismyrealip.com^8 herausfinden.

Bei Spielen im LAN lässt sich die lokale IP Adresse auf folgende Weisen herausfinden:

Doppelklick auf das Verbindungssymbol in der Taskleiste.
(Falls kein Symbol angezeigt werden sollte: "Start" -> "Einstellungen" -> "Netzwerkverbindungen")
Unter "Netzwerkunterstützung" wird die aktuelle IP Adresse angezeigt.

Weitere Informationen gibt es im Online Handbuch: ^7www.lfsmanual.net^8

\use_master
^7Gibt die Art des Servers an

^7LAN^8 - Für Spiele im lokalen Netzwerk. Das Spiel wird nicht in die Serverliste eingetragen, es wird keine Internetverbindung benötigt. Andere Spieler können mit dem Server verbinden, indem sie auf "An Spiel teilnehmen" klicken, "LAN" auswählen und danach die IP Adresse des Servers eingeben.

^7Internet^8 - Die normale Einstellung für Spiele über das Internet. Der Server erscheint in der Serverliste und andere Spieler können einfach durch Klick auf den Servernamen mit dem Spiel verbinden.

^7Hinweis:^8 Falls der Computer über einen Router oder eine Firewall mit dem Internet verbunden ist, müssen für "Internet" und "Versteckt" Spiele die verwendeten TCP und UDP Ports (Standard: 63392) in beide Richtungen geöffnet werden.

Die optimale Spieleranzahl im Mehrspielermodus hängt von der Uploadbandbreite deiner Verbindung ab. Diese ist bestimmt durch den Verbindungstyp und der Geschwindigkeit des Internetanbieters. Wähle aus der Liste die Art deiner Verbindung aus.

\upload_bw
^7Manuelle Eingabe der Verbindungsgeschwindigkeit

Sollte deine Internetverbindung keinem der Standardwerte entsprechen, kannst du hier die Upload Bandbreite in KBit/s eingeben. Dieser Wert hilft dir die richtige Servereinstellung zu wählen, so dass die Upload Kapazität nicht überschritten wird.

Falls die Geschwindigkeit nur in KByte/s bekannt ist, mit 8 multiplizieren.
Bsp.: 32 KByte/s * 8 = 256 KBit/s

Zur Umrechnung von MBit/s zu KBit/s den Wert mit 1024 multiplizieren.
Bsp.: 1 MBit/s / 1024 = 1024 KBit/s

Übliche Geschwindigkeiten: 192 kBit/s (DSL 2000), 1.024 kBit/s (DSL 16000), 5.120 kBit/s (VDSL)

\ded_host
^7Gibt an, ob der Server im dedizierten Modus betrieben wird

Der Server kann im 3D Grafik oder im dedizierten Modus (nur die Übersicht wird angezeigt) betrieben werden.

^7Normaler Server:^8
Der Server wird mit Grafik betrieben. Der Admin kann also das Rennen beobachten und selbst daran teilnehmen.

^7Dedizierter Server:^8
Bei einem dedizierten Server kannst du selbst nicht mitspielen. Der Vorteil gegenüber einem normalen Server ist, dass nur wenig Rechenzeit benötigt wird, da keine 3D Grafik angezeigt werden muss. Ein dedizierter Server kann somit sogar auf einem relativ langsamen PC betrieben werden.

Wenn ein Server über längere Zeit betrieben werden soll, empfielt es sich LFS per Kommandozeile /dedicated=nogfx zu starten. Hierbei wird kein DirectX verwendet, so dass der Server kaum Rechenzeit benötigt und auch auf Computern ohne 3D Grafikkarte ausgeführt werden kann.

\skin_info
Damit LFS automatisch hochauflösende Skins herunterladen kann, musst du LFS World Credit besitzen.

LFS World Credit bedeutet, dass du Geld auf dein LFS Konto eingezahlt hast.

Um diesen Service zu nutzen, musst du eine Zahlung auf ^7www.lfs.net^8 oder ^7www.lfsworld.net^8 vornehmen oder einen LFS Voucher einlösen, den du auf ^7www.live-for-speed.de^8 kaufen kannst.

Für einen Credit von^3 1 £^8 kannst du^3 2000 ^8Skins herunterladen. Der minimale Ladebetrag ist 3 £.

^7So funktioniert es:^8

Du benötigst einen LFS World Credit von mindestens 1 £. Wenn du das erste Mal einen hochauflösenden Skin herunterlädst, wird dir 1 £ von deinem Konto abgezogen und dir stehen 2000 Downloads zur Verfügung. Nachdem du 2000 Skins heruntergeladen hast, wird erneut 1 £ von deinem Konto abgezogen und du kannst wieder 2000 Skins herunterladen.

^7Warnung:^8

Hochauflösende Skins brauchen länger zum Download und es kann zu Rucklern kommen, wenn ein anderer Spieler die Box verlässt. Diese Option wird daher nur empfohlen, wenn du einen guten Computer, eine gute Grafikkarte sowie eine schnelle Internetverbindung besitzt.
\skin_help
Ein ^7Skin^8 ist eine spezielle Grafikdatei, die Farben für die Karosserie oder einen Helm enthält.

LFS verfügt über ein automatisches Skin Download System. Wenn du im S1 oder S2 Modus online fährst, werden die Skins anderer Spieler automatisch auf deinen Computer geladen, so dass du die individuellen Lackierungen ihrer Wagen und Helme sehen kannst.

Wenn du eigene Skins erstellen möchtest, besuche ^7www.lfs.net^8 und lade dir den ^7CMX viewer^8 herunter. Diesen findest du unter "Additional Downloads". Der CMX Viewer ist ein spezielles Programm, das bei der Erstellung von Skins hilft. Vorlagen für Skins sind ebenfalls enthalten.

Nachdem du einen Skin in dein ^7skins^8 Verzeichnis (innerhalb des ^7data^8 Ordners) gespeichert hast, kannst du ihn im Boxenmenü auswählen.

Damit andere Fahrer deine Skins sehen können, besuche ^7www.lfsworld.net^8 und lade deine Skins dort hoch. Danach wird dein ausgewählter Skin automatisch auf die Computer der anderen Fahrer heruntergeladen, wenn sie dich sehen.

\tex_filter
Texturen werden abhängig von ihrer Größe auf dem Bildschirm in verschiedenen Detailleveln dargestellt. Diese Detaillevel werden Mipmaps genannt. Wenn eine Textur kleiner dargstellt wird (wenn also z.B. ein Objekt weit entfernt ist), wird eine kleinere Mipmap verwendet.

Die Textur Filterung beeinflusst die Art, wie zwischen den Mipmaps übergeblendet wird.

^7Keine:^8 Bilineare Filterung - keine Überblendung, es wird direkt zwischen den Mipmaps gewechselt
^71x:^8 Trilineare Filterung - das entstehende Bild wird durch Überblendung von zwei Mipmaps berechnet
^72x - 16x:^8 Anisotropische Filterung - verringert Unschärfe wenn Oberflächen aus einem schrägen Winkel betrachtet werden

Anisotropische Filterung (AF) sieht am besten aus, benötigt aber auch viel Rechenzeit, so dass eine geringere Framerate erzielt wird.

Wenn du AF verwendest, wirst du mit größeren Texturschärfe Einstellungen (den Regler nach rechts ziehen) bessere Ergebnisse erzielen.

Weitere Informationen gibt es im Online Handbuch: ^7www.lfsmanual.net^8
\mip_bias
Diese Optionen beinflussen den Zeitpunkt wann zwischen zwei Mimaps einer Textur umgeschaltet wird (Mipmaps werden in der Hilfe zur Texturfilterung genauer beschrieben).

Bewege den Regler nach links, um schärfere Texturen zu erhalten.
Bewege den Regler nach rechts, um glattere Texturen ohne Flackern/Flimmern zu erhalten.

Wenn eine Oberfläche aus einem schrägen Winkel betrachtet wird (z.B. die Straßenoberfläche), werden die Texturen oftmals zu verschwommen dargestellt. Daher gibt es hier verschiedene Regler für die unterschiedlichen Arten von Oberflächen.

^7Menü/Text:^8 Standardeinstellung -1.0 (wird nicht aus einem schrägen Winkel betrachtet)
^7Objekte/Wände:^8 Standardeinstellung -2.0
^7Gras/Kies:^8 Standardeinstellung -3.0
^7Straßenoberfläche:^8 Standardeinstellung -4.0 (wird meistens aus einem schrägen Winkel betrachtet)

Anisotropische Filterung (AF) ist eine weiter fortgeschrittene Methode um verschwommene Texturen auf schräg betrachteten Oberflächen zu verhindern. Wenn du AF auswählst (Texturen filtern 2x - 16x) wirst du mit höheren Texturschärfe Einstellungen (Regler nach rechts) bessere Ergebnisse erzielen.

Weitere Informationen gibt es im Online Handbuch: ^7www.lfsmanual.net^8
\comp_full
Die Wagen- und Helmskins werden im DDS Format gespeichert. DDS ist ein spezielles, komprimiertes Format, das von Grafikkarten direkt unterstützt wird. Dadurch werden Texturen schneller geladen, es wird weniger Speicher benötigt und eine höhere Framerate erzielt. Die in LFS enthaltenen Skins werden immer in komprimierter Form verwendet.

Skins, die du jedoch selbst erstellst oder in das /skins Verzeichnis herunterlädst, werden im JPG Format gespeichert. Obwohl JPG ein komprimiertes Format ist, kann es nicht direkt von Grafikkarten verwendet werden und ist somit langsamer. Diese Option erlaubt dir, JPG Skins ohne die Umwandlung ins DDS Format zu verwenden, so dass keine Details verloren gehen. In diesem Fall wird der JPG Skin beim Laden dekomprimiert.

Unkomprimierte Skins benötigen mehr Grafikspeicher und verringern die Framerate, bieten aber eine höhere Qualität z.B. für Screenshots.

^7Komprimiert:^8 JPG Skins, die sich im /skins Verzeichnis befinden, werden in das DDS Format umgewandelt und gespeichert.
^7Hoch:^8 JPG Skins, die sich im /skins Verzeichnis befinden, werden beim Laden dekomprimiert und als unkomprimierte Texturen dargestellt.

Weitere Informationen gibt es im Online Handbuch: ^7www.lfsmanual.net^8
