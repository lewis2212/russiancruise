	// - этот символ означает коментарий, он программой не читается.
		Используйте коментарии только на отдельной стоке!!!

	Каждый файл с улицами должен иметь вид TRACK.txt  пример: BL1.txt  SO4R.txt

	Для каждого типа данных используется свой регион, 
	начинающийся с /xxx  и заканчивающийся #xxx

	Улицы надо записывать в порядке возрастания Node от меньшего нода к большему
	Улицы записываются в однустроку указывая начальный и конечный ноды через ;
	как это показано ниже.
	Новая улица начинается со следующего нода после предыдущей

	Зоны боксов и магазина следует указывать в координатах.
	Каждая строчка соответствует виду X;Y отдной точки.
	Первая строчка всегда указывает количество точек, описывающие зону

	ВАЖНО!!! 
	Обход зоны осуществляется из первой к последнейточки по часовой стрелке, не иначе!!!
